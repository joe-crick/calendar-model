weekDayNameFinder

CONSTRUCTOR: Creates a function that returns week day names.

weekDayNameFinder(weekDayNames: Object?= WEEK_DAY_NAMES): Function
Parameters
weekDayNames (Object?= WEEK_DAY_NAMES) Optional. An object with numeric indices 0 - n with values corresponding to the weeks of the month. For example, the default value is: { 0: 'Sunday', 1: 'Monday', 2: 'Tuesday', 3: 'Wednesday', 4: 'Thursday', 5: 'Friday', 6: 'Saturday' };
Returns
Function: A function that will find a week day name when given a number.
Example
const tagesnamen = {
  0: 'Sontag',
  1: 'Montag',
  2: 'Dienstag',
  3: 'Mittwoch',
  4: 'Donnerstag',
  5: 'Fritag',
  6: 'Samstag'
};
const nameFinden = weekDayNameFinder(tagesnamen);
