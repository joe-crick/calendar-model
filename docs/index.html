getDay

CONSTRUCTOR: Returns a day object, which is a JS Date, a formatted string version of the date, and some convenience methods that provide ISO Date, and Week Day Number. Also contains a set of Events for that day. getEvents-: A function that returns a set of events when given a date object. _formatDate: A function that returns a valid formatted date. Defaults to date-fns format function, International date format.
